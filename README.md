KriShi | Real Time Food Delivery Platform

KriShi is a full stack food delivery platform that connects customers with multiple restaurants and enables seamless food ordering with instant order updates. The platform allows users to explore restaurant menus, place orders, make online payments, and track order status in real time.

Restaurant owners can manage incoming orders, update preparation and delivery status, and receive new order notifications instantly through real time communication.

The system is designed to simulate a production level food delivery ecosystem with scalable backend architecture and efficient data handling.

Key Features

• Multi vendor platform where multiple restaurants can manage their menus and orders independently

• Real time order notifications and status updates using Socket.IO

• Secure authentication and authorization using JWT

• Complete order management system for both customers and restaurant owners

• Online payment integration using Razorpay along with Cash on Delivery option

• Responsive and smooth user interface for browsing menus and placing orders

Tech Stack

Frontend: React.js, Redux Toolkit, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB with Mongoose

Real Time Communication: Socket.IO

Payments: Razorpay

Authentication: JWT

What This Project Demonstrates

• Full stack web application development

• Real time system design and event based communication

• REST API design and scalable backend architecture

• Database schema design and efficient data management

• Integration of third party services such as payment gateways
